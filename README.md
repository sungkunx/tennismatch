# 테니스 대진표 생성기

React로 만든 테니스 복식 대진표 생성 앱입니다. 남자/여자 인원, 코트 수, 경기 시간, 경기 타입 배분을 설정해서 대진표를 만들고 공유할 수 있습니다.

## 로컬 실행

```bash
npm install
npm run dev
```

## 배포

이 저장소는 GitHub Pages 배포용 워크플로를 포함합니다.

1. GitHub에 새 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소에 커밋하고 `main` 브랜치로 푸시합니다.
3. GitHub 저장소의 `Settings > Pages`에서 `Source`를 `GitHub Actions`로 설정합니다.
4. `main` 브랜치에 푸시하면 자동으로 빌드 후 배포됩니다.

## 주요 기능

- 남복, 여복, 혼복 대진 자동 생성
- 선수별 경기 수와 파트너/상대 통계 확인
- 시간대별 코트 배정 확인
- 선수 교체 시 영향 분석
- 공유용 텍스트 복사
